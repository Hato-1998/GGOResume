UE5 Top-Down RPG

GAS 기반 액션 RPG

개요

GAS(Gameplay Ability System) 기반 AAA급 코드 아키텍처 학습이 목표인 Top-Down RPG

개인 학습 프로젝트로, GAS의 핵심 추상(Attribute Set, Gameplay Effect, Ability Task)을 직접 구현하며 멀티플레이 복제·UI MVVM 패턴까지 통합 학습

핵심 기여

GAS 기반 어트리뷰트 & 이펙트 시스템

Custom Ability System Component와 Attribute Set 구현으로 Primary(Strength·Intelligence·Resilience·Vigor)와 Secondary(Armor·Crit·Regen) 복제 처리, 화염·번개 등 피해 타입별 저항·디버프 구현

Gameplay Effect와 Attribute Set의 상호작용을 학습하여 서버-클라이언트 복제 환경에서 일관된 게임 상태를 유지하는 방식을 체득했습니다.

스펠 & 패시브 어빌리티 시스템

FireBolt, Electrocute, Arcane Shards, FireBlast 등 액티브/패시브 스펠을 Ability Task 기반으로 모듈화하고, 마나 소모·쿨다운·런타임 단축키 배정 기능 구현

복잡한 스펠 시전 로직을 Async Task와 조합하여 분리된 책임으로 설계했습니다.

Behavior Tree + EQS 기반 적 AI

Warrior(근거리), Ranger(원거리), Elementalist(마법) 타입별 행동 분기와 스펠 시전·미니언 소환 로직을 구현, EQS로 전술적 위치 질의

Behavior Tree와 Environment Query System으로 적 의사결정을 데이터 주도적으로 설계했습니다.

사용 기술

Unreal Engine 5.7 C++ GAS Behavior Tree MVVM
